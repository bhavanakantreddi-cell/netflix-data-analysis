# Netflix Data Analysis

## Overview

This project performs exploratory data analysis (EDA) on the Netflix Movies and TV Shows dataset. The aim is to understand content distribution, trends, and patterns in Netflix data.

## Dataset

The dataset contains information about Netflix content such as:

* Title
* Type (Movie / TV Show)
* Country
* Release Year
* Rating
* Duration
* Genre (listed_in)

## Tools Used

* Python
* Pandas
* NumPy
* Matplotlib
* Seaborn
* Jupyter Notebook

## Analysis Performed

* Distribution of Movies vs TV Shows
* Top countries producing Netflix content
* Year-wise growth of content
* Ratings distribution
* Genre analysis

## Key Insights

* Movies are more in number compared to TV Shows
* The United States produces the highest amount of content
* Netflix content has increased significantly over the years
* Most content falls under TV-MA and TV-14 ratings
* Drama and Comedy are the most common genres

## Conclusion

This analysis provides insights into Netflix content trends, showing growth over time and highlighting popular genres and ratings. It helps understand how content is distributed across different categories.
